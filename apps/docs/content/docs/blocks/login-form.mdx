## Props

The following props can be passed to the component

| Name                | Type       | Default | Description                                                                 |
| ------------------- | ---------- | ------- | --------------------------------------------------------------------------- |
| [`texts`](#texts)   | `object`   |         | Object containing text labels used throughout the form.                     |
| `logosOnly`         | `boolean`  | `false` | If true, only logos are displayed in third-party auth buttons.              |
| `direction`         | `string`   | `ltr`   | Direction of text and UI elements, either left-to-right or right-to-left.   |
| `showError`         | `boolean`  | `false` | If true, an error alert is displayed at the top of the form.                |
| `errorTitle`        | `string`   |         | Title text of error alert.                                                  |
| `errorText`         | `string`   |         | Description text of error alert.                                            |
| `loginType`         | `string`   | `email` | Login identifier type user will use to log in.                              |
| `withResetPassword` | `boolean`  | `true`  | If true, the reset password option is hidden.                               |
| `allowRegister`     | `boolean`  | `true`  | If true, the register option is hidden.                                     |
| `isLoading`         | `boolean`  | `false` | If true, a loading spinner is displayed within the main form submit button. |
| `isGoogleLoading`   | `boolean`  | `false` | If true, a loading spinner is displayed within the Google login button.     |
| `isTwitterLoading`  | `boolean`  | `false` | If true, a loading spinner is displayed within the Twitter login button.    |
| `isGithubLoading`   | `boolean`  | `false` | If true, a loading spinner is displayed within the Github login button.     |
| `viaGoogle`         | `boolean`  | `true`  | If true, Google login option is displayed.                                  |
| `viaGithub`         | `boolean`  | `true`  | If true, Github login option is displayed.                                  |
| `viaTwitter`        | `boolean`  | `true`  | If true, Twitter login option is displayed.                                 |
| `onLogin`           | `function` |         | Function to handle form submission.                                         |
| `onRouteToRegister` | `function` |         | Function to route user to the register page.                                |
| `onForgotPassword`  | `function` |         | Function to handle forgotten password case.                                 |
| `onGoogleLogin`     | `function` |         | Function to handle Google login.                                            |
| `onGithubLogin`     | `function` |         | Function to handle Github login.                                            |
| `onTwitterLogin`    | `function` |         | Function to handle Twitter login.                                           |
| `additionalButtons` | `any`      |         | Additional buttons to add under the login button                            |
| `passwordLength`    | `number`   | `8`     | The allowed length of the password input field                              |
