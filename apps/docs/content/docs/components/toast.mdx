## Usage

<Steps>

<Step>Import `Toaster` component and `useToast` hook</Step>

```js
import { useToast } from "@sikka/hawa/hooks";
import { Toaster } from "@sikka/hawa/toaster";
```

<Step>Get the `toasts` array from `useToast`</Step>

```js
const { toasts } = useToast();
```

<Step>Add the Toaster component</Step>

In your main layout file, add the `Toaster` component. This would be in `_app.tsx`
if you're using Next.js pages directory and `layout.tsx` if you're using Next.js
apps directory.

```tsx
import { Toaster } from "@sikka/hawa/toaster";

export default function MyApp({ children }) {
  return (
    <html lang="en">
      <head />
      <body>
        <main>{children}</main>
        <Toaster />
      </body>
    </html>
  );
}
```

</Steps>

## Props

| Attribute   | Type           | Description                                                                                                                | Default |
| ----------- | -------------- | -------------------------------------------------------------------------------------------------------------------------- | ------- |
| toasts      | array          | The array of toasts. Passed to it from `useToasts()`                                                                       | []      |
| `direction` | `rtl` \| `ltr` | The direction of the toasts. It will appear on the right side if direction is LTR and on the left side if direction is RTL | `ltr`   |
