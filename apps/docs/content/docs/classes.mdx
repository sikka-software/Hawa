The following classes are designed with a specific purpose in mind to simplify your development process by providing ready-to-use styles that can be applied directly to your HTML elements.

For instance, instead of having to import a JSX or React component every time you want to apply a specific style, you can simply use one of these classes. This approach not only makes your code cleaner and easier to read, but also enhances its performance by reducing the number of imports.

## Inline Code

Commonly used to emphaisze code within regular text.

### `inline-code`

<InlinePreview>
  This is <span className="inline-code">inline code</span> created by using a
  the class <span className="inline-code">inline-code</span>.
</InlinePreview>

## Clickable Link

Turn any text into a clickable link which is styled to look like a hyperlink.

### `clickable-link`

<InlinePreview>
  I accept the <span className="clickable-link">terms & conditions</span>
</InlinePreview>

### `link`

<InlinePreview>
  Have a question? <span className="link">Contact Support</span>
</InlinePreview>

## Fade with transparency

Apply a fade effect to the top, bottom, left, or right of an element to create a smooth transition between the content and the background. This can be used on any element not just text

### `hawa-mask-fade-bottom`
